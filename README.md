# ApexFlow
Gamified İngilizce sınav hazırlık uygulaması (Türkçe arayüz). Vite + React, tek dosya build.

## Yayın (Netlify)
Repoyu Netlify'a bağla; netlify.toml ayarları otomatik gelir (build: `npm run build`,
publish: `dist`). Sonraki her push otomatik yayınlanır.

İçerik (kelimeler vb.) ayrı bir content.json ile yönetilir; kod değişmeden büyür.
